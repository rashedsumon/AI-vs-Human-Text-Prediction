# AI vs Human Text Classification System (2026 Dataset Edition)

A production-ready NLP application that predicts whether text samples are human-authored or generated by Artificial Intelligence models.

## Project Framework Properties
* **Dataset Engine:** Kaggle Hub (`alitaqishah/ai-vs-human-text-classification-dataset-2026`)
* **Vectorization:** TF-IDF (N-Gram features)
* **Model Pipeline:** Linear Logistic Regression
* **Frontend UI Matrix:** Streamlit App Cloud
